Current Setup

Genre Program
Window Manager dwm - Excellent lightweight window manager, tiling for efficiency with many pathes availible for extra/changed functionality. Openbox - Up until recently my default WM, lightweight and very customisable floating window manager.
PDF Viewer Zathura - Quick lightweight document viewer with plugin structure and vi-like keybindings. Evince - GTK3 document viewer from the GNOME project.
Web Browser Jumanji - Very fast light web browser built on the gtk2.0 webkit rendering engine with support for plugins and considerable configuration. Similar to dwb, uzbl, luakit. Google Chrome - Fast browser using Goodle's fork of Webkit, Blink, with builtin pdf view and flash plugin. With sync to keep track of bookmarks settings etc across machines.
Editor Vim - I'm only just getting used to this amazingly powerful, lightweight terminal text editor. Amazing potential in the right hands. Sublime Text - Until superceded by Vim, this was my primary editor. Very usful while still looking excellent. Many good plugins and very fast.
Video Player MPlayer - Play anything with no extra fluff. Excellent fast media player, handles anything you throw at it. I use mplayer without any GUI, just from cli.
Music Player MPD - For my library, MPD is searchable, light on resources, can be controlled remotely. GMPC - When some more complex playlist editing is in order, a UI can held; this is just a great front end to MPD.
Document Editor LaTeX - The original and the best document typesetting tool. Take the time to get over the slight learning curve and it will pay off, plus only needs a text editor to do the writing, so choose your favourite. Abiword - For the occasional .doc that is absolutely required Abiword works remarkably well.
Image Viewer Feh - A super lightweight image viewer that can handle a huge range of formats and can set the background wallpaper if you want. Mirage - A good image viewer with a number of editing abilities such as resizing, cropping and colour management.
